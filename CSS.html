Le langage CSS

Le CSS (feuilles de style en cascade1), est un langage informatique qui décrit la présentation des documents HTML. Le CSS a été Introduit en 1995. Maintenant, on est à la version 3.

La balise div

La balise <div> est un bloc, ou un objet rectangulaire qui peut occuper plusieurs lignes. Les attributs de <div> sont: position, margin, padding, width, height. Elle est précédée et suivie d'un saut de ligne.

Margin: permet d'ajouter une marge interne de tous les cotés : haut, bas, droite, gauche.

  • pour siasir la marge, on utilise :
    • margin-top : 10px;
    • margin-bottom : 10px;
    • margin-left : 10px;
    • margin-right : 10px;
  • Si on choisit la même valeur pour les quatre cotés:

    margin : 10px;

  • Si on choisit une valeur (5px) pour haut/bas et une autre valeur (10px) pour droite/gauche :

    margin:5px 10px;

  • Si on choisit une valeur différente pour chaque coté:

    margin:2px 3px 4px 5px; Cela correspond à:

    margin : haut droite bas gauche (dans le sens des éguilles d'une montre).

Padding : permet d'ajouter une marge externe, sa valeur est spécifiée comme l'attribut margin.

La balise <span>

La balise <span> est inline, elle s'inscrit dans le flux du contenu. <span> ne produit pas de saut de ligne. Elle peut s'appliquer au texte dans un paragraphe pour délimiter une phrase.

Comment introduire un style pour une page HTML ?

  • Directement sur l'élément en ajoutant l'attribut style=" "
  • En integrant une balise < style > dans le head. Dans ce cas-là, les règles s'appliquent uniquement sur le document.

    < head >

    < style >

    ...

    </ style >

    </ head >

  • En utilisant un fichier externe style.css

Les selecteurs :

Il y a plusieurs selecteurs en CSS:

  • Le selecteur d'élément (element): en utilisant le nom d'élément comme h1, h2, p,... etc.
  • Le selecteur de classe (.class): il s'applique à tous les éléments qui possèdent la classe. L'attribur class permet d'édentifier un ensemble d'éléments.
  • Le selecteur d'id (#id): il s'applique à un élément unique en utilisant son id (unique).
  • Le selecteur de voisin direct (élément 1+ élément 2): par exemple, p+div s'applique à tous les paragraphes qui sont suivis d'un div. .
  • Le selecteur de voisin indirect ( élément 1~ élément 2) : par exemple, img ~ a s'applique à toutes les images qui sont suivies d'un lien. .
  • Le selecteur d'enfant direct (élément 1 > élément 2): par exemple, div > p s'applique à tous les paragraphes qui descendants immédiats d'un div. .
  • Le selecteur universel (*) : il s'applique à tous les éléments.

Gérer la position des éléments d'une page web

La position peur être spécifiée en utilisant les propriété : left, right, top t bottom. La propriété position spécifie la méthode de positionnement de l'élément, elle peut avoir une des valeurs suivantes :

  • Position relative : la position de l'élément est donnée par rapport à sa position normale.
  • Position absolute : l'élément est positionné par rapport à l'élément qui le contient.
  • Position fixed : l'élément est positionné par rapport à la fenêtre, ce qui signifie qu'il reste toujours à la même place même si la page est défilée.
  • Position stiky : avec la position sticky, on peut maintenir la position d'un élément à un endroit précis dès que celui-ci devient hors de portée. Entendez par là une réaction au scroll. Ce qui sous-entends que cet objet a une position initiale et une position 'en mode sticky'.
  • Position static : l'élément conserve sa position normale dans le flux de la page.

Les unités

  • px : les valeurssont données en pixel.
  • % : les valeurs attribuées de cette façon permettent d'avoir une valeur proportionnelle à la taille de la fenêtre du navigateur ou à l'élément parent de l'élément en question(auquel on attribut cette unité).
  • em : Les valeurs sont données en fonction de la taille de police. L'unité em est simplement la taille de la police de caractères

Les couleurs

Les couleurs des objets (le fond, le texte et les bordures) peuvent être spécifiées en plusieurs façons:

  • En donnant le nom de la ouleur comme : red, green, black, ...
  • En donnant les trois composantes: rouge, vert et bleu sous la forme : rgb (r,g,b) où chaque valeur est comprise entre 0 et 255.
  • En donnantt les composantes rouge, vert et bleu en utilisant le système héxadécimal sous la forme : #rrggbb, où les composantes rr, gg et bb sont comprises entre 00 et FF (héxadécimal) ce qui correspond à 255 en base 10.

Les bordures

La bordure peut être ajoutée en utilisant la propriété border:

border: épaisseur style couleur;

border: 1px solide black;

Nous pouvons aussi spécifier les attributs de border un par un, comme suit:

border-width: 1px;

border-style: solide;

border-color: black;

La bordure peut avoir d'autres attribut comme radius qui permet d'arrondir les angles et les cotés d'un rectangla par exemple.

border-radius: 5px;

border-radius:75px/12px;

Les pseudo-classes

Comme hover qui permet de modifier un élément quand il est survolé de la souris.

.Rouge:hover{

background-color:#00FF00;

}

Les pseudo-éléments

Comme before qui permet de faire une modification avant l'élément :

.p::before{

content:"**";

}
